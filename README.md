PayPal Zahlungstransaktion 

Funktionalität: 
  - Empfänger auswählen
  - Betrag auswählen
  - Währung auswählen inklusive Währungsumrechnung
  - Automatische Gebührenberechnung bei abweichender Währung 
  - Ergebnis Seite: Transaktion erfolgreich/Transaktion fehlgeschlagen

Technologien: 
  Datenbank: MySQL
  Backend: Java, Spring Boot
  Frontend: HTML, CSS, JavaSript, Bootstrap
